# Central TI

Sistema desenvolvido em Java com Spring Boot para gerenciamento da equipe de TI.

## Tecnologias

- Java 21
- Spring Boot 3
- SQL Server 2022
- Spring Data JPA
- Thymeleaf
- Maven

## Funcionalidades previstas

- Login
- Cadastro de usuários
- Painel da equipe de atendimento
